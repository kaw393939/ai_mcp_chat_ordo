# Chapter 7 — Privacy and Compliance: GDPR, CCPA, and Ethical Data

## Abstract

Data collection without privacy safeguards is surveillance. This chapter covers GDPR, CCPA, and the engineering discipline of privacy-by-design: collecting only what you need, storing it securely, and deleting it when it's no longer needed.

## Key Principles

- **Data minimization**: collect only what you need for a stated purpose
- **Purpose limitation**: use data only for the purpose it was collected for
- **Right to deletion**: users can request their data be deleted
- **Consent**: users must actively consent to data collection

## Chapter Checklist

- Do you collect only the data you need (data minimization)?
- Can users request deletion of their data?
- Is user consent explicit, not pre-checked?
