# Chapter 8 — Data-Informed Decision Making: Not Data-Driven

## Abstract
Data informs decisions; it does not make them. This chapter covers the distinction between data-driven (the data decides) and data-informed (the data informs human judgment), and the failure modes of each approach.

## The Distinction
**Data-driven**: "The data says Feature A performs better, so we ship Feature A." (Risk: the data is correct but the context is wrong — Feature A performs better short-term but damages brand trust long-term.)

**Data-informed**: "The data says Feature A performs better. Given our strategic priorities and our understanding of customer needs, we decide to ship Feature A — with the caveat that we'll monitor long-term effects." (The human provides context the data cannot.)

## Chapter Checklist
- Are data insights accompanied by contextual interpretation?
- Do you have guardrail metrics that prevent optimizing one metric at the expense of others?
- Is your team data-informed, or data-driven (know the difference)?
