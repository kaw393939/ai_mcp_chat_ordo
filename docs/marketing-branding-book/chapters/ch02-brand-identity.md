# Chapter 2 — Brand Identity Systems: Visual Language as Promise

## Abstract
A brand identity is a system of visual and verbal elements that communicate a consistent promise. This chapter covers Wally Olins's brand philosophy, identity system components, and the relationship between brand tokens and design tokens.

## Brand System Components
| Element | Purpose |
|---------|---------|
| Logo | Recognition mark |
| Color palette | Emotional association |
| Typography | Voice and personality |
| Imagery style | Narrative consistency |
| Tone of voice | Verbal personality |
| Spacing and layout | Structural consistency |

These elements map directly to design tokens — a brand system IS a design token system at the strategic level.

## Chapter Checklist
- Does your brand have a documented identity system?
- Do design tokens implement brand decisions (not just aesthetic preferences)?
- Is your brand voice documented and consistently applied?
